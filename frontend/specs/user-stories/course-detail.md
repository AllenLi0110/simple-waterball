# 使用者故事：課程詳情頁

## 故事描述

**作為** 一個學生  
**我想要** 查看課程詳情和章節  
**以便於** 我可以了解課程內容和觀看影片

## 驗收標準

### AC1: 課程資訊顯示
- [ ] 應該顯示課程標題
- [ ] 應該顯示課程描述
- [ ] 應該顯示章節列表

### AC2: 章節列表功能
- [ ] 章節應該按順序顯示
- [ ] 點擊章節應該展開影片列表
- [ ] 選中的章節應該高亮顯示
- [ ] 第一個章節應該自動選中

### AC3: 影片播放功能
- [ ] 點擊影片應該在播放器中顯示
- [ ] 播放器應該載入正確的影片 URL
- [ ] 應該顯示影片標題和描述
- [ ] 應該顯示影片時長（如果有）

### AC4: 導航功能
- [ ] 返回按鈕應該導航回上一頁
- [ ] 點擊課程卡片應該能進入詳情頁

### AC5: 錯誤處理
- [ ] 課程不存在時應該顯示錯誤資訊
- [ ] API 錯誤時應該顯示錯誤提示

## 技術規格

### API 端點
- `GET /api/courses/{id}` - 獲取單個課程詳情

### 響應格式
```json
{
  "success": true,
  "message": "Successfully retrieved course",
  "data": {
    "id": 1,
    "title": "課程標題",
    "description": "課程描述",
    "chapters": [
      {
        "id": 1,
        "title": "章節標題",
        "videos": [
          {
            "id": 1,
            "title": "影片標題",
            "videoUrl": "https://example.com/video.mp4",
            "duration": 600
          }
        ]
      }
    ]
  }
}
```

### 組件
- `CourseDetailPage` - 課程詳情頁組件
