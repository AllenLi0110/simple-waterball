# 使用者故事：課程列表

## 故事描述

**作為** 一個學生  
**我想要** 查看所有可用的課程  
**以便於** 我可以選擇我想學習的課程

## 驗收標準

### AC1: 課程列表顯示
- [ ] 首頁應該顯示所有課程
- [ ] 每個課程應該顯示在卡片中
- [ ] 課程卡片應該包含：標題、副標題、描述、圖片、價格、按鈕

### AC2: 課程卡片樣式
- [ ] 推薦課程應該有特殊的視覺樣式（黃色高亮）
- [ ] 非推薦課程應該有預設樣式
- [ ] 所有課程卡片應該有懸停效果

### AC3: 課程卡片互動
- [ ] 點擊課程卡片應該跳轉到課程詳情頁
- [ ] 點擊按鈕也應該跳轉到課程詳情頁
- [ ] URL 應該包含課程 ID

### AC4: 載入狀態
- [ ] API 請求期間應該顯示載入提示
- [ ] 載入完成後應該隱藏載入提示

### AC5: 錯誤處理
- [ ] API 錯誤時應該顯示錯誤資訊
- [ ] 錯誤資訊應該包含有用的除錯資訊

## 技術規格

### API 端點
- `GET /api/courses` - 獲取所有課程列表

### 響應格式
```json
{
  "success": true,
  "message": "Successfully retrieved course list",
  "data": [
    {
      "id": 1,
      "title": "課程標題",
      "subtitle": "課程副標題",
      "description": "課程描述",
      "imageUrl": "/images/course_0.png",
      "priceText": "價格資訊",
      "buttonLabel": "按鈕文字",
      "isFeatured": true
    }
  ]
}
```

### 組件
- `HomePage` - 首頁組件
- `CourseCard` - 課程卡片組件
